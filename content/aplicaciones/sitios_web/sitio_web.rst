=========
Sitio Web
=========

Daeris incluye un módulo con herramientas de creación de sitios web que ayudarán a integrar todas tus operaciones
minoristas en la aplicación. Daeris puede ayudarte a crear un sitio web comercial vibrante para tu empresa. Puedes publicar
tus blogs, foros, eventos y productos en tu sitio web, lo que ayudará a los visitantes a buscar las cosas que necesitan
de tu empresa.
